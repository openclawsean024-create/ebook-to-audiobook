# 電子書轉有聲書 - 專案規格

## 目標
將上傳的電子書自動轉換為有聲書，可選擇多種 AI 語音

## 功能需求

### 1. 電子書上傳
- 支援格式：PDF, EPUB, TXT, MOBI, DOCX
- 顯示書籍資訊（標題、作者、頁數）
- 進度顯示

### 2. 文字轉語音 (TTS)
- 支援多種語音模型
- 調整語速、音調、音量
- 支援中文、英文、日文等多語言

### 3. 語音選擇
- 不同說書人風格
- 自然語音模型
- 預覽功能

### 4. 輸出
- 下載 MP3 檔案
- 支援分段下載

## 技術方案

### 後端
- Python (Flask/FastAPI)
- ebook2txt (文字提取)
- Edge TTS / Google TTS / Azure TTS

### 前端
- React + Vercel
- 拖放上傳
- 播放預覽

### API
- Edge TTS (免費，自然)
- Azure TTS (高品質)
- ElevenLabs (最自然，需付費)

## 語音模型推薦

| 模型 | 費用 | 品質 | 語言 |
|------|------|------|------|
| Edge TTS | 免費 | ★★★★☆ | 多語言 |
| Azure TTS | 免費額度 | ★★★★★ | 多語言 |
| ElevenLabs | 付費 | ★★★★★ | 多語言 |
| VITS | 開源 | ★★★☆☆ | 中文 |
